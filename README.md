# Rime
小狼毫五笔输入法

## 项目来源

拷贝自: [小狼毫](https://github.com/rime/weasel)与 [Rime 五笔](https://github.com/myshiqiqi/rime-wubi).

## 安装步骤

将 `WeaselSetup.exe` 中 `用户文件夹` 设定为 `\rime-wubi`, 通过 `WeaselDeployer.exe` 部署之.

## (部分) 捷径

- <code>Ctrl + `</code> 菜单,
- `help` 帮助,
- ... 

繁体五笔输入应先开启扩展字库.
